ActivateAbilityByClass

Attempts to activate the ability that is passed in. This will check costs and requirements before doing so. Returns true if it thinks it activated, but it may return false positives due to failure later in activation.

Inputs

Ability Class

GSCGameplay Ability Class Reference

Gameplay Ability Class to activate

Allow Remote Activation

Boolean

If true, it will remotely activate local/server abilities, if false it will only try to locally activate abilities.

Outputs

Activated Ability

GSCGameplay Ability Object Reference

The Gameplay Ability that was triggered on success

Return Value

Boolean

bSuccess Returns true if it thinks it activated, but it may return false positives due to failure later in activation.

ActivateAbilityByTags

Attempts to activate a **single** gameplay ability that matches the given tag and DoesAbilitySatisfyTagRequirements(). It differs from GAS ASC TryActivateAbilitiesByTag which tries to activate *every* ability, whereas this version will pick a random one and attempt to activate it. Returns true if the ability attempts to activate, and the reference to the Activated Ability if any.

Inputs

Ability Tags

Gameplay Tag Container Structure

Set of Gameplay Tags to search for

Allow Remote Activation

Boolean

If true, it will remotely activate local/server abilities, if false it will only try to locally activate abilities.

Outputs

Activated Ability

GSCGameplay Ability Object Reference

The Gameplay Ability that was triggered on success

Return Value

Boolean

bSuccess Returns true if it thinks it activated, but it may return false positives due to failure later in activation.
